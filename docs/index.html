PineCMS内容管理系统

一个GO语言和Sqlite开发的内容管理系统

快速开发

以标签化的方式创建网站, 一杯茶的功夫即可. 只要会html就可以完全自主的搭建一个称心的网站.

自主框架

自主开发Web框架, 最少依赖打包, 可以让您的网站文件更小更易迁移.

高性能

基于Go语言开发, 性能至上. 页面支持静态化, 让您的站点达到访问静态网站一样的性能.
