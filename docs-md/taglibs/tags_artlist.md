# 内容列表标签
|功能| 描述|
| :------------- |:-------------|
| 作用      | 获取指定文档列表 |
| 属性      |    |  
| 可嵌套 | 可与channelartlist 嵌套, 内部自动接收channelartlist迭代返回的typeid参数.  |
| 备注 | 数据表: 各个模型的内容   |   


# 标签暴露变量

|变量| 描述|
| :------------- |:-------------|
| autoindex      | 迭代时索引值 |
| field | 一条数据记录结构 |

# 模型文档属性

|属性| 描述|
| :------------- |:-------------|
| Id      | 广告ID |
| Name      | 广告名称   |  
| Image | 图片地址  |
| LinkUrl | 点击广告需要跳转的地址    |
